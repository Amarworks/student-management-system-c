# Student Management System (C)

A command-line based Student Management System written in the C programming language.
This project demonstrates fundamental concepts of C programming through a modular and structured implementation.

The application allows users to manage student records including adding, viewing, searching, updating, and deleting entries. Data can be stored and retrieved using file handling.
[CLI based Student Management System written in C]

---

## Project Overview

This project was created as a learning exercise to strengthen core programming concepts such as structured programming, modular design, and data management using C.

The program operates through a command-line interface and maintains student records using structures and file storage.

---

## Features

* Add new student records
* Display all stored students
* Search students by ID
* Update student information
* Delete student records
* Sort students based on marks
* Persistent storage using files

## Compilation

Compile the program using GCC from the project root directory:

```
gcc src/main.c src/student.c -o sms
```

---

## Running the Program

After compilation, run the executable:

Linux / macOS

```
./sms
```

Windows

```
sms.exe
```
